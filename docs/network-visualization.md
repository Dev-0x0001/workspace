# Network Visualization Tool

## Overview

A comprehensive network visualization tool with host discovery, port scanning, and OS fingerprinting capabilities.

## Features

1. Local network scanning
2. Interactive host exploration
3. Port scanning functionality
4. OS fingerprinting
5. Visual network mapping

## Usage

1. Clone the repository
2. Install dependencies
3. Run `npm start` to launch the application
4. Access the network visualization interface

## Installation

```bash
npm install
npm start
```
